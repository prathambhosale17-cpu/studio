# **App Name**: AadhaarVerify AI

## Core Features:

- Aadhaar Simulation: Simulate Aadhaar document scanning or uploading, including necessary data extraction for verification.
- Verification Status: Display a clear, real-time verification status based on the simulation.
- Fraud Indicators: Show potential fraud indicators identified during the analysis. The LLM acts as a tool to extract only important information from this stage.
- Verification History: Maintain a log of all verification attempts, including timestamps, status, and identified indicators, if applicable.

## Style Guidelines:

- Primary color: Deep teal (#008080) for trust and security.
- Background color: Light grayish-teal (#E0EEEE), providing a clean, professional backdrop.
- Accent color: Sea green (#3CB371) for positive confirmations and highlights, creating a sense of reassurance.
- Body and headline font: 'Inter', sans-serif, for a modern, neutral and easily readable experience.
- Use clear, universally recognized icons to represent different verification states (e.g., success, failure, pending).
- A clean, card-based layout to display verification details and history in an organized manner.
- Subtle animations during the scanning and verification processes to provide user feedback.
